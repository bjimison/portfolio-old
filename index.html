FashionCents

I developed FashionCents as a solo project at General Assembly. I used React.js, Node.js, Express.js, and MongoDB. The purpose of the application is to provide an online marketplace for clothing designers where they can receive upvotes on their products. To view the full project, please visit my github. My biggest accomplishment was the type-search functionality via the input box just below the navbar. When the user types in a search filter, the value is extracted and checked to see if the database contains any posts with titles containing the same sequence of letters.

SuiteHealth was a four-member group project I helped develop during my Full Stack Web Development Immersive bootcamp. We developed Suite Health using HTML, CSS, and Javascript for the front-end, Django for the back-end, and a PostgreSQL database. Suite Health is a comprehensive health application where users can discover new meal plans and workouts based on their fitness goals. See the code snippet below or view the full project in my github to see the full code base. My biggest triumph during this project came by way of successfully rendering newly created workouts and meals via Ajax calls and the new ES6 Template Literals.
